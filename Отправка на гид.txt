1. Подготавливаем все новые и измененные файлы:

Bash
git add .
(Точка означает: "захватить вообще все изменения во всех файлах и папках").

2. Создаем "сохранение" с описанием (коммит):

Bash
git commit -m "Добавил систему регистрации пользователей"
(Здесь обязательно меняй текст в кавычках! Пиши коротко и понятно, что именно ты сейчас сделал. Например: "Исправил ошибку сокета", "Сверстал профиль" и т.д. Это история твоего проекта, по которой ты потом сможешь легко ориентироваться).

3. Отправляем всё на сервер GitHub:

Bash
git push
(Так как в первый раз мы уже связали твой компьютер и GitHub длинной командой git push -u origin main, теперь Git всё запомнил. Достаточно писать просто git push).
